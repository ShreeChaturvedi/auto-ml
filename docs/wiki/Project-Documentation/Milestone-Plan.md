# Milestone Plan

| Milestone | Outcome |
| --- | --- |
| Project foundation | Monorepo, frontend/backend skeletons, project storage, upload flow, and initial data exploration. |
| Data and context workflow | Dataset parsing/profiling, document ingestion, SQL querying, natural-language query path, and RAG/search groundwork. |
| Agentic preprocessing | LLM workflow streaming, preprocessing plans, notebook-backed generated code, approval/retry/interrupt flows. |
| Feature engineering and training | Feature pipelines, derived datasets, Docker-backed Python execution, model training records, notebook UI. |
| Experiments and evaluation | Model leaderboard, comparison, SHAP/error analysis, tuning, generated reports, benchmark/eval suites. |
| Deployment and operations | Deployment dashboard, serving container lifecycle, predict proxy, API keys, logs, stats, drift checks, runtime health. |
| Final hardening | UX polish, authentication/ownership, tests, benchmarks, expo assets, and current wiki documentation. |

Day-to-day execution is tracked in GitLab issues, merge requests, and sprint artifacts. This wiki keeps the durable milestone summary rather than duplicating the full issue board.
